Welcome to Anime Database!

Our mission is to provide anime fans with a simple, free, and convenient method of accessing information about their favorite animes.

Fans who want to learn accurate information about many of their favorite animes often must resort to conducting multiple search queries and website visits, resulting in a very inconvenient user experience.

Our Solution: Anime Database

We used Jikan API to create a website that allows users to learn more about their favorite animes through our custom search feature. Jikan API is an open-source PHP & REST API that scrapes data about thousands of different animes from MyAnimeList. You can learn more here.

We also feature the top 5 airing animes, in addition to all airing animes, on our homepage, allowing the user to browse our site and potentially find new animes to explore. Users are able to access the folowing data: average score, rating, synopsis, producers, studios, genres, and character information.

We Hope You Enjoy!
